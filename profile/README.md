# BEE Korea의 GitHub Organization

BEE Korea는 신학교에 갈 수 없는 교회 지도자들에게 현장에서 성경 훈련을 제공하여, 제자를 양육하고 지도자를 재생산하는 ‘이동식 신학교’입니다.

이 GitHub Organization은 BEE Korea의 웹사이트 및 LMS 소스코드를 저장하고, 수정하며, 배포하기 위해 BEE Korea ICT 팀원들이 같이 협력하는 곳입니다.

+ 2019-2020: 외주 개발자 분이 파이썬 Django 기반 LMS (beekorea.org/lms) 개발. AWS Codecommit & CodeDeploy, RDS, EC2, 등 AWS 기반 인프라, 툴링 사용.
+ 2025: 손**, 강** 형제의 합류로 낙후된 웹사이트 및 LMS 점검, 현대화 (버전 업그레이드), DB 정리. AI 에이전트 (Claude Code, OpenAI Codex)로 코딩 없이 BEE 본부에서 사이트를 수정할수 있도록 셋업. AWS Codecommit 지원 중단으로 GitHub로 코드 이전.
